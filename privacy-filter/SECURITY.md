# Security Policy

For a more in-depth look at our security policy, please review OpenAI's
[Coordinated Vulnerability Disclosure Policy](https://openai.com/policies/coordinated-vulnerability-disclosure-policy/).

Please report vulnerabilities through OpenAI's
[Bugcrowd program](https://bugcrowd.com/openai) or by emailing
[disclosure@openai.com](mailto:disclosure@openai.com).

OpenAI's PGP key and security contact metadata are published in
[security.txt](https://openai.com/security.txt).
